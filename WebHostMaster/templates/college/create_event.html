Event Guidelines

Event Types:
  • SPORTS: Physical competitions like basketball, running, etc.
  • ACADEMIC: Knowledge-based competitions like quizzes, debates, etc.
  • CREATIVE: Artistic competitions like writing, art, music, etc.
  • TECHNICAL: Skills-based competitions like coding, robotics, etc.
  • GAMING: Video game or board game competitions.
Event Formats:
  • Individual Event: Participants compete on their own.
  • Team Event: Teams of 5 members compete together.
Important: Once created, the format (team/individual) of an event cannot be changed.

After Creation

After creating an event, you'll be able to:

  • Register participants/teams for the event
  • Record results and rankings
  • View event details and participants
  • Track points and update rankings
