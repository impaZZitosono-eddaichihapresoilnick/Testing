Tournament Structure

College Tournament Format:
  • Each tournament can have multiple events (up to 5 recommended)
  • Events can be for teams or individuals
  • Team events require full team participation
  • Individual events allow solo participation
  • Points are awarded based on rankings in each event
  • Overall tournament rankings track both team and individual performances
Event Types Examples:
  • SPORTS: Basketball, Soccer, Table Tennis
  • ACADEMIC: Quiz, Debate, Programming
  • CREATIVE: Art, Music, Writing
  • TECHNICAL: Robotics, Engineering, Science
  • GAMING: Chess, Video Games, Board Games

After Creation

After creating a tournament, you'll need to:

  1. Add events to the tournament
  2. Define scoring systems for events
  3. Manage team and individual registrations
  4. Record results for each event
  5. Track overall tournament rankings
Important: Events must be created separately after tournament creation.
